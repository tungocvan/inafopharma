== Description ==
Create a quick quote request for any trade. Easy management with separate drag and drop steps for each quote.
Manage multiple quotes at the same time, manage quote request list, customer information by phone, email...
After receiving the quotation will automatically send notification email. And can manage multiple currencies.

Tạo yêu cầu báo giá nhanh chóng với mọi ngành nghề. Quản lý dễ dàng với các bước kéo thả riêng cho từng báo giá. 
Quản lý nhiều báo giá cùng lúc, quản lý danh sách yêu cầu báo giá, thông tin khách hàng diện thoại, email...
Sau khi nhận báo giá sẽ tự động gửi mail thông báo. Và có thể quản lý nhiều loại tiền tệ.

**Video**

https://www.youtube.com/watch?v=cCRg-hI-1jY

**Features**

* Create unlimited quotes
* Convenient drag and drop
* Send and receive email when customers send quotation
* Manage customer quote request
* Mark quote request: viewed, contacted, completed
* Manage multiple currencies
* Allow to translate many different languages
* Allows developers to add_action template, add_filter email content

**Tính năng**

* Tạo báo giá không giới hạn
* Kéo thả tiện lợi
* Gửi nhận mail khi khách hàng gửi báo giá
* Quản lý yêu cầu báo giá của khách hàng
* Đánh dấu yêu cầu báo giá: đã xem, đã liên hệ, hoàn thành
* Quản lý nhiều loại tiền tệ
* Cho phép dịch nhiều ngôn ngữ khác nhau
* Cho phép nhà phát triển add_action template, add_filter nội dung gửi email

== Installation ==

1. Tải thư mục `quick-quotes-wpshare247` vào đường dẫn `/wp-content/plugins/`
2. Kích hoạt từ menu **Plugins** (**Plugins > Installed Plugins**).

Tìm **Cấu hình QQFS** hoặc **Configure QQFS** menu trong Admin Menu.
